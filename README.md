# Pace Calculator

A lightweight, mobile-responsive running pace calculator. Calculate your race pace from a goal time, or estimate your finish time from a target pace.

## Features

- **Goal Time → Pace**: Enter a goal finish time to calculate required pace
- **Pace → Finish Time**: Enter a pace to see estimated finish times
- **Mile/Kilometer toggle**: Automatically converts pace when switching units
- **Split times**: View cumulative splits at each mile or kilometer
- **Dark/Light mode**: Respects system preference with manual toggle
- **Race distances**: 1 Mile, 5K, 10K, 15K, 10 Miles, Half Marathon, Marathon

## Usage

Open `index.html` in any browser. No build step or dependencies required.

## Tech

Single HTML file with inline CSS and JavaScript. No frameworks, ~8KB total.
